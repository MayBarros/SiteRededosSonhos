# 🌙 Rede dos sonhos

Uma plataforma web desenvolvida em React que permite aos usuários compartilhar sonhos, experiências e pedidos de ajuda, criando uma comunidade de apoio, empatia e conexão.

---

## 📖 Sobre o projeto

A **Rede dos sonhos** é um site onde pessoas podem publicar sonhos e receber apoio, conselhos ou incentivo de outros usuários.

A ideia é criar um espaço seguro e acolhedor para expressão e colaboração.

---

## ✨ Funcionalidades

* 📝 Publicação de sonhos e relatos
* 💬 Comentários e interações entre usuários em chat.
* ❤️ Sistema de apoio (likes ou reações)
* 🔍 Visualização de sonhos de outras pessoas

---

## 🛠️ Tecnologias utilizadas

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* API REST / Sqlite / Node.js

---

## 🚀 Como rodar o projeto

### Pré-requisitos

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)
